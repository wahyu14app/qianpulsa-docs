# Dokumentasi Fitur Ekosistem

Folder ini berisi definisi fitur dalam ekosistem QianPulsa. Setiap file mendefinisikan:
1. Konsep fitur.
2. Kapabilitas tiap aplikasi (Admin, Seller, Store) terhadap fitur tersebut.
3. Keterkaitan dengan entitas data di database.

Struktur folder:
- /docs/features/customer.md
- /docs/features/seller.md
- /docs/features/product.md
- /docs/features/transaction.md

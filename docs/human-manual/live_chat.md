# Fitur: Layanan Live Chat

Dokumen ini mendefinisikan mekanisme fitur live chat di Storefront.

## 1. Integrasi
- Fitur ini terintegrasi dengan **Support API** pihak ketiga.

## 2. Tanggung Jawab
- **Admin**: Hanya berfungsi sebagai pengawas teknis integrasi.
- **Seller**: Layanan pelanggan (pembeli) sepenuhnya adalah tanggung jawab Seller di masing-masing Toko. Admin tidak terlibat dalam penyelesaian masalah pembeli.

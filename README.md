# QianPulsa Ecosystem

QianPulsa adalah platform SaaS PPOB Multi-Tenant (B2B2C) yang memungkinkan pengelolaan toko produk digital secara mandiri.

Arsitektur sistem terdiri dari:
- **Core App**: Backend API pusat.
- **Admin App**: Panel kontrol super-admin.
- **Seller App**: Dashboard manajemen toko (B2B).
- **Store App**: Etalase toko untuk pelanggan (B2C).

Dokumentasi detail arsitektur dan panduan pengembangan tersedia di direktori `docs/`.
